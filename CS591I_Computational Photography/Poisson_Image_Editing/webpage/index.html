Introduction

Image editing tasks are related to either global changes (color/intensity, corrections, filters, deformations) or local changes confined to a selection. Poisson image editing technique mainly focuese on achieving local changes that users can manually select a target region and manipulate different modifications in a seamless and effortless way. In this project, I implemented the Poisson image editing techniques introducted in this paper by P. Pérez, M. Gangnet, A and Blake and achieved several interesting tools about local image edting, such as replacing or mixing the target region from the source image with the indicated location on the destination image, or altering the illumination or the texture of the given image.

In this project, I implemented the three algorithms described in the paper including Seamless Cloning Algorithm, Mixing Gradients Algorithm and Selection Editing Algorithm. And I also accomplished the GUI design that provides users a convenient and direct way to employ thses useful image editing tools.

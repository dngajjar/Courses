Seamless Cloning

Simply cutting and pasting method always results in some noticeable seams around the cutting boundaries. Introducing the seamless cloning technique, we can achieve the compliance of source and destination borders. This tool can conceal undesirable image features or to insert new elements in an destination image in a flexibility and convenient way.

In Seamless Cloning algorithm, the guidance field is defined by the following equation, which is wholly dependent on the gradients of the source image.

Employing the guidance field in the solution functions for each points on the selected area and solving the function set, we can get the new pixels values.

In this algorithm, I generated two approaches for users to define the target area on the source image as following:

1. Square Label

In this method, users can define the editing area by labeling four points on the source image than can be consisted as a square and contain the target object, and also indicate the location of pasting the selected area on the destination image.

2. Mask Label

Also, in order to precisely extract the object on the source image, I manually created the mask. Users can select the corresponding pair of points on the source image and the destination image to determine the pasting location.

Comparing to the simply cutting and pasting approach, we can easily see that the seams become less noticable and the editing region ahieved a sort of uniform appearance.

3. Bad Results

However, there are several limitations about this algorithm. The first one is that the algorithm assumes that the color of the source image is close to that of the destination image. And this tool can ahieve the function of tweaking the color on the source image to that of the destination image. Here is a bad result about this disadvantage. We cut and paste a white bear to a blue background image, then we get a blue bear.

Another drawback is that under some specific situation, it may require precise alignment of the selected region and the destination image.

All the images used in the project are from google images.

***************************************************************
Use GUI
***************************************************************
1. Input "GUI_new" on the Matlab command line
2. Choose the editing tool, source and destination image
3. Get the resulting images on the given or default output directory

***************************************************************
Direct calling the function 
***************************************************************
1. Choose the corresponding function by running the 'main_*****.m' file, 
	***** indicates the specific function
	
***************************************************************
function files
***************************************************************
1. All programmed function files are saved in the 'functions' folder
2. The usage of parameters are illustrated in each file.

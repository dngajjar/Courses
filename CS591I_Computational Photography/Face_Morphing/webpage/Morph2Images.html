Morphing Two images

 

In the first part to this project, I achieved the function of generating several frames that represents the process of morphing from one image to another. The calculation process can be expressed as following:

1. Pick 43 pairs of corresponding points from two images, and compute the mean points.

2. Define the triangulation on the mean points.

3. For each frame with different wrap_frac and dissolve_frac that changes with time:

a). Compute the average shape based on the wrap_frac;

b). For each triangle in the average shape:

- Get the affine matrix projection to the corresponding triangles in each images to the average shape;

- For each pixel in the generated images (for both input images) with the average shape, find the corresponding points in each input images and set value to average.

c). Combine these two generated images with the average shape based on the dissolve_frac.

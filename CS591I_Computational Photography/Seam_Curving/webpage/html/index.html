Introduction

 

Seam carving is one of the most popular image cutting and image retargeting methods. This powerful algorithm is able to remove a set of pixels from the input images that avoid cutting the major content in order to to generate the output images. Seam carving method was introduced by Avidan and Shamir in 2007. Since its advantages in simplesity, processing efficiency and memory efficiency, this method has been implemented in a lot of applications, including in Adobe Photoshop, the Liquid Rescaleplugin for GIMP and some online websites.

In this project, we aim to understand the procedures of conducing the seam carving and implement this method in order to create our target images. In this webpage, several tabs are presented on the left that link to several sections of this projects.

Different Energy Function

 

There are several important measures mentioned in this paper that can be employed as the energy function to guide the seam carving.

1). Imag Gradient:

- Using the "gradient" function in Matlab conducted in the 3-channel of color images to get the gradient value for x, y direction, get the absolut value for the addition of these two direction and compute the average gradient value for each pixel of three channels.

- Using Sobel Operator to find the vertical and horizontal gradients.

2). Image entropy:

Using a 9*9 window to compute the entropy of the image and add the value to the e1 function mention in part 1).

By comparsion, I found out that the running time for energy function 1) is much lower than energy function 2).

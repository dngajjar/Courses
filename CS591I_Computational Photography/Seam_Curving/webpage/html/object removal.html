Object Removal

 

In this experiment, we aim to remove marked target object in the original image using the seam carving algorithm. In my opinion, this goal can be achieved according to the following procedures:

1). Label the object that is going to remove from the image;

2). Change the energy of the pixels belonging to the marked objects to a negative value;

3). Provide the seam carving algorithm with target number of seams that are going to remove;

4). The algorithm will find out the smallest total energy of pathes in the energy map. Since in step 2) we have labeled those useless pixels with negative values, the shortest pathes will conclude these points.

5). The algorithm will carve these determined seams that cover the target object.

The work of labeling the object can be done using a mask with the contour of the object. I did the work by choosing four point in the image that build a rectangle, which will contain the removal region. Then set the energy of pixels in these region as -10. And the algorithm will determine how many seams will be removed according to the size of the marked region. Then the algorithm will repeatly remove a seam with the lowest total energy until all the marked pixels are removed.
